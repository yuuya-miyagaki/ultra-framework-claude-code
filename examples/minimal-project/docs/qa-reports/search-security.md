# セキュリティレビュー

## 対象

- 変更内容: 検索 UI と mock index API
- 脅威面: 入力文字列、結果スニペット、社内限定アクセス

## 確認項目

- 認証 / 認可: 社内 VPN 前提で外部公開なし
- 入力検証: 検索入力は文字列長制限と空値処理を実装
- データ露出: スニペットは保存済み文書の公開対象のみ表示
- 秘密情報: 秘密情報の直書きなし

## OWASP Top 10 チェック

| # | カテゴリ | 該当 | 備考 |
|---|---|---|---|
| A01 | アクセス制御の不備 | n/a | VPN 前提、個別認証なし |
| A02 | 暗号化の失敗 | n/a | 機密データの暗号化対象なし |
| A03 | インジェクション | 確認済 | 入力は文字列長制限とサニタイズを実施 |

| A06 | 脆弱なコンポーネント | n/a | 依存パッケージは最新 |
| A07 | 識別と認証の失敗 | n/a | VPN 前提 |

| A10 | SSRF | n/a | 外部リクエスト機能なし |
