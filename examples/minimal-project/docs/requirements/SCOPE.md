## 対象範囲

- 検索入力フォーム
- 結果一覧表示
- 出典スニペット表示
- ドキュメント種別フィルター

## 対象外

- 権限ベースの詳細アクセス制御
- ドキュメント自動取り込みパイプライン
- 検索ログ分析ダッシュボード

## 将来の検討事項

- 権限連携の導入（次フェーズ）
- 実検索基盤への差し替え

## 依存関係

- 既存の社内ドキュメント一覧 CSV
- mock search index

## 優先度（MoSCoW）

| 優先度 | 項目 |
|--------|------|
| **Must** | 検索フォーム、結果一覧、出典スニペット |
| **Should** | ドキュメント種別フィルター |
| **Could** | 検索候補表示 |
| **Won't** | 権限制御、自動取り込み、分析ダッシュボード |
