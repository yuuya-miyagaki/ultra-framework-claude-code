# 要件定義書

## プロジェクト概要

- プロジェクト名: 社内ナレッジ検索アシスタント
- 担当者: AI Platform Team
- 状態: Dev 完了 / handover 準備中

## 課題定義

- 解決したい課題: 社内ドキュメントの所在が分散しており、必要な情報に辿り着くまで時間がかかる
- なぜ今やるか: 問い合わせ件数が増えており、検索導線の標準化が必要になった

## ユーザー

- 主対象ユーザー: 社内オペレーション担当
- 副対象ユーザー: 新任メンバー

## ゴール

- ビジネスゴール: 社内 FAQ 問い合わせを削減する
- ユーザーゴール: 必要な資料と該当箇所を 1 回の検索で見つける
- 成功指標: 検索成功率 80% 以上、主要クエリの 2 秒以内応答

## 機能要件

- FR-001: キーワードで社内ドキュメントを検索できる
- FR-002: 結果一覧にタイトル、要約、出典スニペットを表示する
- FR-003: ドキュメント種別で結果を絞り込める
